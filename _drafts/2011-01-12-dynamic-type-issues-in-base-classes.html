Today I ran into a little issue with C#'s dynamic type and I got a nice answer on stack overflow.

If you have a this class hierarchy:

public class Generic {
public T Save(T obj) {
// it's not abstract
}
}

public class A {
protected abstract dynamic NonGeneric { get; }

public void Save() {
NonGeneric.Save(this);

}
